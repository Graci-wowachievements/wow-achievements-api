achievementsApi.php Readme 21/02/2013

Purpose

The achievementsApi.php library is meant to access Blizzard's API to get achievements data for World of Warcraft. It is used in the wowAchievements.net website.

Description

There is an Interface AccèsApiBlizzardInterface and a Class AccèsApiBlizzardClass.

Usage

You have to define a class that extends AccèsApiBlizzardClass and define in it the functions described in point 2 "// 2-Fonctions nécessaires à la fonction ApiMajPersonnages, à redéfinir en fonction de votre appli et de votre base de données" to be able to call the function ApiMajPersonnages that updates your characters according to Blizzard Api.

The available functions to access the Api are defined in the AccèsApiBlizzardInterface.

Two data arrays are available : Hosts and Locales, defined in the AccèsApiBlizzardClass.
